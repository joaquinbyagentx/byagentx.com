🚀 QUICK START:

1. Copy to project:
   cp -r assets/ /path/to/byagentx.ai/public/

2. Import in HTML:
   <link rel="stylesheet" href="/assets/buttons/payment-buttons.css">

3. Add button:
   <button class="payment-button premium size-lg">Start Now</button>

4. Add badge:
   <img src="/assets/badges/secure-checkout-badge.svg" alt="Secure" 
        width="200" height="60">

   Open examples/checkout-example.html in browser

📞 TROUBLESHOOTING:

Issue: Buttons not showing correctly
→ Verify CSS file loaded (check DevTools Network tab)
→ Clear browser cache
→ Check console for errors

Issue: Icons not appearing
→ Ensure SVG file path is relative to HTML location
→ Check viewBox attributes in SVG
→ Test with direct image tag first

Issue: Not responsive on mobile
→ Add viewport meta: <meta name="viewport" content="...">
→ Test with actual device, not just browser emulation
→ Check media queries in CSS

Issue: Accessibility warnings
→ Run through axe DevTools or WAVE
→ Test keyboard navigation (Tab key)
→ Test with screen reader (NVDA/JAWS)
