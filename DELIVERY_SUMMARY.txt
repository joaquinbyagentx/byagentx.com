================================================================================
FLUJO DE PAGO
================================================================================

1. Cliente visita https://byagentx.ai#pricing
2. Selecciona plan y hace clic en "Buy Now"
3. Redirigido a payment.html?plan=PLAN_ID
4. Completa formulario (nombre, email, país, código postal)
5. Ingresa datos de tarjeta en Stripe Elements
6. Hace clic en "Pay $X.XX"
7. Página envía token a /api/create-payment
8. Backend procesa pago en Stripe:
   - Si setup fee > 0: Carga uno-tiempo para setup
   - Si monthly fee > 0: Crea suscripción mensual recurrente
9. Backend retorna success con redirect URL
10. Cliente redirigido a index.html?success=true
11. Cliente tiene acceso inmediato al dashboard

INSTALACIÓN & DEPLOYMENT
================================================================================

REQUISITOS:
- Node.js v16+
- npm o yarn
- Cuenta en Stripe.com (gratuita)

INSTALACIÓN LOCAL:
1. cd /Users/josecortes/.openclaw/workspace/byagentx
2. npm install
3. cp .env.example .env
4. Agregar claves Stripe a .env:
   STRIPE_PUBLISHABLE_KEY=pk_live_...
   STRIPE_SECRET_KEY=sk_live_...
5. npm start
6. Abrir http://localhost:3000/payment.html?plan=whatsapp-bot

DEPLOYMENT A PRODUCCIÓN:
- Heroku: heroku create app-name && heroku config:set STRIPE_SECRET_KEY=...
- Railway: Similar (desde dashboard)
- Self-hosted: VPS + PM2 o systemd

Ver STRIPE_SETUP.md para instrucciones detalladas.
